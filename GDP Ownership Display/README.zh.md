# GDP所有权显示

![Thumbnail](thumbnail.png)
![Thumbnail 2](thumbnail2.png)

## 功能

- 在国家面板快速显示所有国家的所有权比例
- 突出显示你的国家所持有/被持有的建筑等级
- 分别显示海内外GDP所有权占比
- 悬浮在GDP占比数值可显示按拥有GDP数额降序排序的完整所有权清单

## 支持的语言

- 简体中文
- 英语
